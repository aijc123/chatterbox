# 我给 B 站直播间做了个弹幕助手：独轮车、自动跟车、评论区增强一条龙

如果你经常挂在 B 站直播间，应该很熟悉这些场景：

- 想刷一组固定弹幕，但一条一条复制太麻烦。
- 直播间突然开始复读某句话，想跟一句，又不想一直盯着聊天区。
- 右侧弹幕区信息太挤，礼物、进场、弹幕混在一起，看久了眼睛累。
- 想知道自己粉丝牌关联的房间里，有没有禁言、屏蔽、风控或者主播注销。
- 某些弹幕总是发不出去，想快速试试替换写法，但又不想每次手动改。

于是我做了一个 userscript：**B站独轮车 + 自动跟车**。

它是一个运行在 B 站直播间页面里的弹幕助手，可以通过 Tampermonkey 或 Violentmonkey 安装。安装后，直播间右下角会出现一个 `弹幕助手` 按钮，点开就能使用独轮车、自动跟车、Chatterbox Chat、粉丝牌禁言巡检、替换规则、AI 规避候选、同传和烂梗库等功能。

项目地址：

## 它能做什么？

### 1. 独轮车：把重复弹幕交给脚本

独轮车功能适合提前准备一组弹幕，让脚本按节奏循环发送。

你可以把多句话一行一条放进模板里，设置发送间隔，然后让它自动轮流发送。它也支持随机间隔、随机颜色、随机字符和长弹幕拆分，尽量减少“手动复制粘贴”的疲劳感。

比较适合：

- 直播活动应援。
- 固定口号循环。
- 需要保持节奏但不想一直手动发的场景。

当然，还是建议控制频率。直播间是大家一起看的地方，刷屏爽归爽，别把别人淹了。

### 2. 自动跟车：直播间开始复读时，自动跟一句

很多直播间都会突然进入“复读模式”：某句话被很多人连续发送，气氛一下就起来了。

自动跟车功能会观察最近一段时间内的弹幕。如果同一句话被多人重复发送，并且达到了你设置的阈值，它就会自动跟上一句。

为了避免变成无脑刷屏，它内置了冷却时间和多人判断，还提供了几个预设：

- `稳一点`
- `正常`
- `热闹`

你也可以展开高级设置，自己调观察窗口、触发条数、唯一用户要求和冷却时间。

如果有些固定刷屏账号你不想让它们带节奏，也可以加入融入黑名单，自动跟车会忽略这些人的弹幕。

### 3. Chatterbox Chat：换一个更清爽的右侧评论区

原生 B 站直播间右侧评论区信息量很大，礼物、进场、弹幕、醒目留言全挤在一起，有时候看久了会累。

Chatterbox Chat 可以接管右侧评论区，显示更清爽的聊天视图。它会读取 B 站直播 WebSocket 事件，展示弹幕、礼物、醒目留言、舰队、进场、关注、点赞和分享等内容，同时保留 DOM 兜底解析。

常用操作也放在消息旁边：

- 偷弹幕
- +1
- 复制
- 发送前确认

如果你喜欢折腾样式，也可以用自定义 CSS 调整显示效果。

### 4. 粉丝牌禁言巡检：一键看看相关房间状态

粉丝牌房间多了之后，有时候想知道哪些房间状态异常：有没有禁言、屏蔽、风控、主播注销，或者接口无法确认。

粉丝牌禁言巡检会读取当前账号粉丝牌关联的直播间，并把结果分类展示：

- 限制
- 无法确认
- 主播已注销
- 正常

巡检不会自动发送弹幕。结果可以复制，方便自己记录或者反馈问题。

如果你使用“直播间保安室”同步功能，也可以选择同步摘要。但这个功能是可选的，而且只应该同步巡检摘要或选定规则，不会要求你上传 cookie、csrf、localStorage 或完整接口响应。

### 5. 替换规则和影子屏蔽候选

有些弹幕不是明确报错，而是看起来发出去了，实际上别人看不到。这种情况很像“影子屏蔽”。

插件会做广播验证：发出弹幕后，观察 WebSocket 或页面里是否真的出现了这条弹幕。如果疑似没有广播，它会记录观察结果，并给出一些候选改写。

默认模式是“只给候选”：

- 不自动重发。
- 不替你做决定。
- 只把可能的改写列出来。

你可以选择复制，或者填入输入框自己确认后再发。

如果你明确开启“自动重发”模式，并且同时开启 AI 规避，脚本才会尝试 AI 改写并自动重发。

### 6. 同传、翻译和烂梗库

插件还保留了同传、翻译和烂梗库相关能力。

同传部分接入 Soniox 语音识别，可以把语音识别结果或翻译结果发送到直播间。烂梗库则适合快速搜索、复制、发送常用梗，也可以根据直播间弹幕挖掘待贡献候选。

这些功能都不是强制开启的，你可以按自己的直播间习惯慢慢试。

## 怎么安装？

先安装一个用户脚本管理器：

- Tampermonkey：https://www.tampermonkey.net/
- Violentmonkey：https://violentmonkey.github.io/

然后打开 Greasy Fork 安装页：

https://greasyfork.org/zh-CN/scripts/574939-b%E7%AB%99%E7%8B%AC%E8%BD%AE%E8%BD%A6-%E8%87%AA%E5%8A%A8%E8%B7%9F%E8%BD%A6

点击安装，确认脚本管理器弹窗。

安装完成后，打开 B 站直播间：

```text
https://live.bilibili.com/
```

页面右下角会出现 `弹幕助手` 按钮。

## 权限和隐私简单说一下

这个脚本只匹配 B 站直播间页面：

```text
*://live.bilibili.com/*
```

它会用脚本管理器的本地存储保存你的设置，比如：

- 弹幕模板
- 自动跟车配置
- 本地替换规则
- Chatterbox Chat 偏好
- 粉丝牌巡检缓存
- 影子屏蔽观察记录

它会访问 B 站接口和 WebSocket，用来读取直播间事件、发送弹幕、获取粉丝牌相关房间信息。

可选功能里，Soniox 用于同传/语音识别；直播间保安室用于同步摘要或选定规则。这些功能都可以不开。

请注意：不要在反馈问题、截图或导出配置时公开 cookie、csrf token、账号密钥、localStorage dump、私人规则或私有同步地址。

## 适合谁用？

我觉得它比较适合这几类人：

- 经常挂 B 站直播间，想轻松发固定弹幕的人。
- 喜欢跟直播间复读节奏，但不想一直盯弹幕的人。
- 对原生右侧评论区观感不满意，想换个更清爽聊天视图的人。
- 有很多粉丝牌，想快速看房间状态的人。
- 经常遇到弹幕发不出去，想维护替换规则的人。

如果你只是偶尔看直播，也可以只开 Chatterbox Chat 或手动发送功能，不一定要用自动发送。

## 已知限制

这个插件依赖 B 站页面结构、接口和 WebSocket 数据。B 站一改页面或接口，插件就可能需要跟着修。

另外：

- 自动发送可能被 B 站限流或风控。
- 自动跟车不是越猛越好，阈值太低会打扰别人。
- 影子屏蔽检测是启发式判断，不能百分百准确。
- 替换规则和 AI 规避不能保证每条弹幕都能通过。
- 同传效果受麦克风、浏览器和 Soniox 服务影响。
- 主要测试环境是桌面浏览器 + Tampermonkey/Violentmonkey。

## 最后

这个插件的目标不是“更猛烈地刷屏”，而是把直播间里一些重复、繁琐、容易误触的操作收进一个更可控的面板里。

你可以把它当成一个弹幕工具箱：只开自己需要的功能，慢慢调到适合自己和直播间的节奏。
