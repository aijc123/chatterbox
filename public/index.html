把高频操作收得更优雅。

从刷屏节奏到评论区观感，常用功能都保持清楚、稳定、可控。二次元不靠堆贴纸，而是藏在弹幕、光线和状态反馈里。

独轮车循环发送

多套模板、每行一句、按间隔发送，支持随机颜色、随机间隔、随机字符和长弹幕自动拆分。

自动跟车

观察短时间内重复出现的弹幕，达到阈值后自动跟一句，内置稳一点、正常、热闹三档预设。

Chatterbox Chat

接管右侧评论区，显示 iMessage 风格气泡，支持礼物、醒目留言、舰队事件和自定义 CSS。

粉丝牌禁言巡检

读取粉丝牌关联房间并分类展示限制、无法确认、主播注销和正常结果，可复制报告或同步摘要。

替换与 AI 规避

支持云端和本地替换规则，发送失败时可检测敏感词并尝试替换后重试，让队列更少卡住。

同传和烂梗库

接入 Soniox 语音识别，可自动发送识别或翻译结果，也能搜索、复制和发送常用梗。

Install

三步进入直播间。

仍然是熟悉的用户脚本安装方式，只是落地页更像一个认真打磨过的产品入口。

  1. 安装脚本管理器

    先安装 Tampermonkey 或 Violentmonkey，用来管理浏览器 userscript。

  2. 打开 Greasy Fork

    进入脚本页面并点击安装，浏览器会自动交给脚本管理器确认。

  3. 进入 B 站直播间

    打开直播间，点击右下角「弹幕助手」。需要接管评论区时，在「设置」里开启 Chatterbox Chat。
